Hello C World

My first blog will discuss the very basic c code that prints the string Hello C World.

Build Environment

Linux based OS with gcc installed

If not you can get OS image from here https://ubuntu.com/#download

Now you can install gcc using the following command

$ sudo apt install gcc

Getting started:

open gedit and type the following program

xxxxxxxxxx
#include <stdio.h>
int main()
{
    printf("Hello C World\n");
    return 0;
}

Now save it as Hello.c and compile with the following command

xxxxxxxxxx
$ gcc Hello.c -o Hello

Now run the program with the command

xxxxxxxxxx
$ ./Hello

You will get the following output

xxxxxxxxxx
Hello C World
